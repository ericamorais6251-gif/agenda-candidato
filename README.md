# 🏍️ LuziLeva
O comércio de Luziânia na sua porta

Fundado por Érica Morais · CNO

## Sobre
App de delivery modelo iFood
para o comércio local de Luziânia - GO

## Perfis
- Cliente: comprar nas lojas
- Lojista: cadastrar e vender
- Motoboy: fazer entregas
- Admin: painel da Érica Morais

## Stack
- React Native (iOS + Android)
- Node.js + NestJS
- PostgreSQL + Redis
- Mercado Pago API
- Google Maps GPS
